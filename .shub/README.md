# ShubCoGen Template

## Usage

Use `deploy.sh` to deploy your project on GitHub.

## Optional arguments

```sh
./deploy.sh [-a] [-m message] [-h]
  -a: Accept all
  -m: Tag message
  -h: Help
```

### -a
Accept all suggestions, and deploy project.

### -m
Set tag message.

### -h
Show help.
